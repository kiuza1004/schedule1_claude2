# Claude Schedule 2

A TypeScript-based app for managing text data with GitHub Actions and GitHub Pages deployment.

## Features
- CRUD operations for managing text data.
- GitHub Actions for CI/CD.
- Deployment via GitHub Pages.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kiuza1004/schedule1_claude2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd schedule1_claude2
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Scripts
- `npm run build`: Compile the TypeScript code.
- `npm start`: Run the application.
- `npm test`: Run tests.
- `npm run lint`: Lint the code.

### Deployment
The app is deployed via GitHub Pages. Ensure the `gh-pages` branch is set up for deployment.
